Man Series - Skin Minimalism
Science-backed, clinical men's skincare designed for Indian men.
About the Project
Man Series is a minimalist skincare web application showcasing 3 multi-functional products formulated specifically for Indian men's skin. The design focuses on premium aesthetics, dynamic animations, and clinical transparency.
Features
Modern & Premium UI: Glassmorphism, smooth animations, and a responsive dark theme.
Dynamic Reactor: An interactive "Molecule Reactor" showcasing active ingredients and their benefits based on skin concerns.
Responsive Design: Optimized for both mobile and desktop screens.
Cart System: Interactive slide-out cart functionality (frontend only).
Tech Stack
HTML5
CSS3 (Vanilla, Custom properties, Animations)
JavaScript (Vanilla DOM manipulation)
Deployment
This project is ready to be deployed on Vercel as a static HTML/CSS/JS site.
